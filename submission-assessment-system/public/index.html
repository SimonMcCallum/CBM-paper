System Features

  • Novelty Detection: Identifies novel vs. standard content using LLM-powered semantic analysis
  • Context-Aware: Compares submissions against course materials (lecture notes, textbooks, etc.)
  • Intelligent Question Generation: RAG-based retrieval for standard content, LLM-generated for novel content
  • Confidence-Based Marking: Students provide confidence levels (1-5) for calibrated assessment
  • Oral Assessment Questions: Auto-generated questions to verify deep understanding
  • Multiple LLM Providers: Support for Claude, Gemini, GPT, and local LLM servers
  • Comprehensive Reporting: Novelty reports, oral question sheets, and detailed analytics
  • Canvas LTI Integration: Seamless integration with Canvas LMS (coming soon)
